# TASK-P01-004: แสดงเลขอารบิกแทนเลขไทยใน Salah Time Webapp

**Status:** Ready (Leantime #TBD)
**Priority:** High
**Depends on:** TASK-P01-002 (Done)

## Objective

เปลี่ยนการแสดงผลตัวเลขทั้งหมดใน webapp จากเลขไทย (๐๑๒๓๔๕๖๗๘๙) เป็นเลขอารบิกปกติ (0123456789)

## Root Cause

ใน `script.js` มีฟังก์ชัน `toThaiDigits()` ที่ถูกเรียกใช้กับทุกตัวเลขที่แสดงผล ได้แก่:
- เวลาละหมาด (HH:MM)
- countdown timer
- วันที่ Hijri
- วันที่ Gregorian (พ.ศ.)

## Fix

ใน `script.js` ให้เปลี่ยนฟังก์ชัน `toThaiDigits` ให้ return input ตรงๆ โดยไม่แปลง:

```js
function toThaiDigits(input) {
  return String(input);
}
```

ไม่ต้องแก้ไขฟังก์ชันอื่น ไม่ต้องแก้ไฟล์อื่น

## Files

- `script.js` — แก้เฉพาะฟังก์ชัน `toThaiDigits` เท่านั้น
