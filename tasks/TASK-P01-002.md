## API

`GET https://api.aladhan.com/v1/timingsByCity?city=Bangkok&country=Thailand&method=1&school=1`

- No API key required
- Extract: `data.timings` (Fajr, Sunrise, Dhuhr, Asr, Maghrib, Isha)
- Extract: `data.date.hijri` (day, month.en, year)

## Logic

- **Current prayer**: last prayer whose time ≤ now → highlight green
- **Next prayer**: first prayer time > now → countdown
- **Countdown**: update every second, display HH:MM
- **Buddhist year**: Gregorian year + 543
- **Settings**: city input → localStorage → refetch API
