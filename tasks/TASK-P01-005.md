# TASK-P01-005: Dark Mode Toggle

**Status:** Ready (Leantime #TBD)
**Priority:** High
**Depends on:** TASK-P01-002 (Done)

## Objective

เพิ่มปุ่ม Dark Mode ใน webapp ให้ผู้ใช้สลับ light/dark ได้ และจำค่าไว้ใน localStorage

## UI

- ปุ่ม toggle อยู่มุมบนซ้ายของ card (ข้างๆ โลโก้)
- Icon: 🌙 (dark) / ☀️ (light)
- Dark mode: พื้นหลัง #1a1a2e, card #16213e, text #eee, highlight row #2d4a22

## Logic

- default = light mode
- toggle → เปลี่ยน class `dark` บน `<body>`
- บันทึกค่าใน `localStorage.getItem('salah_theme')` = `'dark'` หรือ `'light'`
- โหลดหน้าใหม่ให้ restore theme จาก localStorage

## Files

- `style.css` — เพิ่ม `.dark` CSS variables/overrides
- `script.js` — เพิ่ม theme toggle logic
- `index.html` — เพิ่มปุ่ม toggle ใน header
