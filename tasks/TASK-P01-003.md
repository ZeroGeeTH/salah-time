## Objective

No code changes needed. This task configures the repository only.

## Steps

1. Enable GitHub Pages on repo `ZeroGeeTH/salah-time`
2. Set source: branch `main`, folder `/` (root)
3. Verify the site loads at `https://zerogeeth.github.io/salah-time/`

## How to Enable (GitHub CLI)

```bash
gh api repos/ZeroGeeTH/salah-time/pages \
  --method POST \
  -f build_type=legacy \
  -f source.branch=main \
  -f source.path=/
```
